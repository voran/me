Odroid X2

This server is actually an odroid x2.

It hosts all my web services, and acts as a print, media, and VPN server.

Hardware

  • CPU: 4x1.7GHz ARMv7 cores clocked @ 1.9 GHz
  • RAM: 2GB
  • 6xUSB
  • UART
  • HDMI
  • Audio in & out
  • Storage
    • 32 GB SD Card as /dev/mmc0blk1
    • 3 TB WD External HDD as /dev/sda2

Software

  • Arch Linux ARM
  • Nginx + Rails
  • CUPS*
  • Samba*
  • OpenSSH
  • OpenVPN

* Behind Firewall
