Database Explorer (DBE) is a web-based database administration tool written in PHP. It supports MySQL, PostgreSQL, Firebird, SQLite, Oracle, and Microsoft SQL Server. DBE can detect primary key, foreign key, and null constraints and can import and export data in SQL, XML, and CSV.

Share Scanner

0.00

Share Scanner is a Samba share browser for GNU/Linux. It uses the PyGTK library and integrates well with the GNOME desktop. Share Scanner indexes network shares on the local network and presents the user with a list of files, their size, and their location on the network. Share Scanner can download multiple files at once.
